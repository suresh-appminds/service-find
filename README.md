# Find Near By Ambulances or Doctors - JOIN Coding Challenge - Frontend (React) Backend (Node JS)
<img src="illustration.jpeg" width="100%" alt="Find Near By Ambulances or Doctors">

## Context

Accident cases are increasing more nowadays. So this app needs to display the list of all nearby ambulance services and doctors based on location with one click.


## Product Requirements

As a product ower:
- [ ] I want to add, edit, update and delete the ambulance and doctors 
- [ ] I want to see a list of ambulance and doctors
- [ ] I want to see the first 10 records, with the ability to - paginate (10 cases per page).
- [ ] I want to see a total ambulance and doctors in app.
- [ ] For each record I want to see:
  - [ ] Title
  - [ ] Description
  - [ ] Location
  - [ ] Picture of the ambulance or doctors, if available
- [ ] I want to see a loading state until the list is available.
- [ ] I want to see an error state if the list is unavailable.
- [ ] I want to see an empty state if there are no results.

As a user:
- [ ] Need to show all list doctors and ambulance 
- [ ] Need to add filter for type (doctors or ambulance)
- [ ] For each record I want to see:
  - [ ] Title
  - [ ] Description
  - [ ] Location
  - [ ] Distance from current location
  - [ ] Picture of the ambulance or doctors, if available
- [ ] I want to see a loading state until the list is available.
- [ ] I want to see an error state if the list is unavailable.
- [ ] I want to see an empty state if there are no results.

## Your Mission

- Need to create two page
    - Landing page for user
    - list page for admin (Authindation not required)
